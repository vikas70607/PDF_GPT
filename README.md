## Introduction
The Custom Knowledge ChatGPT with LangChain is an innovative project that combines the power of OpenAI's ChatGPT and LangChain technologies to enable seamless chat interactions with PDF documents. This project aims to make information retrieval from PDFs more accessible and user-friendly, allowing users to have natural language conversations with the system and receive relevant information from PDFs.

## Features
- **Custom Knowledge ChatGPT:** Utilize OpenAI's advanced ChatGPT technology to enable dynamic and interactive conversations with the system.
- **LangChain Integration:** LangChain integration allows for real-time processing of PDF documents, making it possible to extract knowledge and answers directly from PDF files.
- **Natural Language Interaction:** Users can communicate with the system in natural language, making the interaction more intuitive and user-friendly.
- **PDF Upload and Processing:** Users can upload PDF documents, and the system will automatically extract the relevant information, facilitating a seamless chat experience.
- **Contextual Understanding:** The system is designed to retain context throughout the conversation, providing more accurate and context-aware responses.

## Installation
Follow the steps below to set up the Custom Knowledge ChatGPT with LangChain project:

1. Clone the repository: `git clone https://github.com/your_username/custom-knowledge-chatgpt.git`
2. Navigate to the project directory: `cd custom-knowledge-chatgpt`
3. Install the required dependencies: `pip install -r requirements.txt`
